# 📝 React To Do List

A modern To Do List application built with **React**, **Tailwind CSS**, and **React Icons**.  
This project was created to practice React fundamentals including state management, conditional rendering, local storage, dark mode, and CRUD operations.

---

## ✨ Features

- ➕ Add new tasks
- ✏️ Edit existing tasks
- 🗑️ Delete tasks with confirmation dialog
- 🔍 Search tasks instantly
- 💾 Save tasks automatically using Local Storage
- 🌙 Dark Mode / Light Mode
- 📱 Responsive UI
- 🎨 Reusable Button Component
- 🪟 Edit task using Modal

## 🚀 Installation

Clone this repository

```bash
git clone https://github.com/yourusername/react-todo-list.git
```

Go to project folder

```bash
cd react-todo-list
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```
